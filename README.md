# Pulse VPN

Pulse VPN — это легкий и современный VPN-клиент на базе React Native и Expo (bare workflow), разработанный для работы со списками подписок VLESS. 

---

## ⚙️ Как работает приложение

Приложение автоматизирует весь процесс настройки и подключения к VPN-серверам:

1. **Загрузка и кэширование списков**
   При старте приложение обращается к вашему URL-адресу подписки и скачивает актуальный список серверов. Если сеть временно недоступна, приложение автоматически загрузит последний сохраненный в памяти рабочий кэш.

2. **Парсинг VLESS-конфигураций**
   Загруженные данные разбираются парсером. Он находит ссылки формата `vless://`, извлекает из них параметры подключения (хост, порт, UUID, протоколы шифрования и передачи данных) и формирует список доступных серверов.

3. **Проверка скорости и выбор лучшего**
   Приложение отправляет проверочные запросы (пробы) ко всем серверам из списка, измеряет задержку отклика (пинг) и автоматически сортирует их. Вы всегда видите самые быстрые и стабильные серверы вверху списка.

4. **Запуск нативного VPN-туннеля**
   При нажатии кнопки «Подключить» приложение запрашивает у системы разрешение на создание VPN-соединения и запускает нативный системный мост:
   - **Android**: Инициализирует системную службу `VpnService`.
   - **iOS**: Использует службу `NetworkExtension` (Packet Tunnel Provider).
   
   *Примечание: Для реального шифрования и перенаправления трафика используется ядро sing-box (библиотеки `libbox.aar` для Android и `Libbox.xcframework` для iOS).*

## 🚀 Быстрый старт (Разработка)

### 1. Установка зависимостей
Убедитесь, что у вас установлены Node.js 20+ и JDK 17 (для Android) / Xcode + CocoaPods (для iOS).

```bash
# Установка NPM-пакетов
npm ci

# Установка CocoaPods (только для macOS/iOS)
cd ios && pod install && cd ..
```

### 2. Запуск проекта
Приложение использует нативные модули VPN, поэтому его **нельзя** запустить в обычном Expo Go. Запуск производится через Development Client:

```bash
# Запуск Metro Bundler
npm run start

# Запуск на Android (эмулятор или устройство)
npm run android

# Запуск на iOS (симулятор)
npm run ios
```

## 📦 Сборка релизных версий

### Android (APK)
```bash
npm run build:android:apk
```
Готовый файл сохранится в: `dist/android/app-release.apk`

### iOS (IPA)
```bash
APPLE_TEAM_ID=<ВАШ_TEAM_ID> npm run build:ios:ipa
```
Готовый файл сохранится в: `dist/ios/App.ipa` (требуется платный аккаунт разработчика Apple с поддержкой NetworkExtension).

## 🛠️ Настройка и решение проблем

* **Своя подписка**: Ссылку на список серверов по умолчанию можно изменить в файлах `src/utils/constants.ts` и `app.config.ts`, либо поменять прямо в интерфейсе настроек приложения.
* **Ошибка «PulseVpnBridge is unavailable»**: Возникает при попытке запуска в приложении Expo Go. Используйте сборку dev-client (`npm run android` / `npm run ios`).
* **Интеграция sing-box**: Для маршрутизации трафика необходимо добавить скомпилированную библиотеку `libbox` в проект:
  - Android: положите `libbox.aar` в classpath приложения.
  - iOS: подключите `Libbox.xcframework` в Xcode.
